## ✨ 核心特性

### 🎯 全方位清理
- **Windows临时文件** - 清理系统和用户临时文件
- **Windows更新缓存** - 释放更新留下的垃圾
- **浏览器缓存** - 支持Chrome、Edge、Firefox、Opera、Brave
- **软件开发缓存** - 清理node_modules、pip cache、npm cache、huggingface cache等
- **AI工具缓存** - 清理Cursor、VSCode、Trae、Ollama缓存
- **聊天软件缓存** - 清理微信、QQ、Telegram、Discord缓存
- **媒体软件缓存** - 清理Spotify、网易云音乐缓存
- **系统日志** - 清理Windows日志和错误报告
- **大文件智能分析** - 找出占用空间的大文件
- **重复文件检测** - 基于MD5哈希的重复文件查找

### 🛡️ 安全机制
- **回收站保护** - 文件移到回收站，可恢复
- **白名单保护** - 永不清理系统关键文件
- **受保护扩展名** - .sys、.dll、.exe等文件类型不删除
- **危险路径警告** - Windows目录、System32等敏感区域拒绝删除
- **管理员权限检测** - 自动提示权限问题
- **可视化界面** - 所见即所得，心里有数

### 🎨 用户体验
- **现代化GUI** - 基于tkinter的美观界面
- **分类标签页** - 清晰的分类展示
- **实时进度** - 扫描过程实时反馈
- **磁盘监控** - 实时显示C盘使用情况

## ⚙️ 配置说明

所有配置都在 `config.py` 文件中，可以根据需要调整：

### 大文件阈值
```python
LARGE_FILE_THRESHOLD = 100  # 单位：MB
```

### 重复文件最小大小
```python
DUPLICATE_MIN_SIZE = 10  # 单位：MB
```

### 扫描驱动器
```python
SCAN_DRIVES = ['C:\\']  # 可以添加其他驱动器
```

### 排除文件夹
```python
EXCLUDE_FOLDERS = [
    'C:\\Windows\\System32',
    'C:\\Program Files\\WindowsApps',
    # 可以添加更多要排除的文件夹
]
```

## 🛡️ 安全说明

### 保护机制
1. **白名单保护** - 系统关键文件永不清理
2. **受保护文件夹** - 重要系统文件夹禁止删除
3. **权限检查** - 自动检测和请求必要权限
4. **清理预览** - 清理前显示将要删除的内容

### 清理级别
- **临时文件** ✅ 安全
- **浏览器缓存** ✅ 安全（会清除登录状态）
- **Windows更新缓存** ✅ 安全
- **开发缓存** ⚠️ 注意（会清除node_modules等，需重新安装）
- **系统日志** ⚠️ 注意（可能影响故障诊断）

### 建议
- 首次使用建议先查看扫描结果，了解将要清理的内容
- 重要项目的node_modules建议不要清理
- 清理前确保没有正在运行的程序
- 定期备份重要数据

## 🔧 常见问题

### Q1: 提示权限不足怎么办？
**A:** 右键点击 `启动清理工具.bat`，选择"以管理员身份运行"

### Q2: 扫描很慢怎么办？
**A:** 
- 大文件扫描和重复文件检测比较耗时
- 可以在 `config.py` 中调整扫描范围
- 排除不需要扫描的文件夹

### Q3: 清理后能恢复吗？
**A:** 
- ✅ 文件会移到回收站，可以恢复
- 如需彻底释放空间，请清空回收站

### Q4: 能清理其他盘吗？
**A:** 
- 可以修改 `config.py` 中的 `SCAN_DRIVES`
- 例如：`SCAN_DRIVES = ['C:\\', 'D:\\']`

### Q5: node_modules被清理了怎么办？
**A:** 
- 进入项目目录重新安装依赖
- npm项目：`npm install`
- Python项目：`pip install -r requirements.txt`

## 📊 性能提示

### 加速扫描
- 排除不需要扫描的文件夹
- 提高大文件阈值
- 提高重复文件最小大小
- 使用SSD硬盘

### 最佳实践
- 定期清理（建议每月一次）
- 关闭所有程序后清理
- 清理后重启计算机
- 配合Windows自带的磁盘清理工具使用

## 🎯 清理效果预期

根据测试，一般可以释放：
- **临时文件**: 500MB - 5GB
- **Windows更新**: 1GB - 10GB
- **浏览器缓存**: 500MB - 3GB
- **开发缓存**: 5GB - 50GB（如果有很多项目）
- **系统日志**: 100MB - 1GB

**总计预期**: 2GB - 70GB（取决于使用习惯）
