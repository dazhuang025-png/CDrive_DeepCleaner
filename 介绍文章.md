# 我为什么自己写了一个C盘清理工具

## 背景

作为一个开发者，我的C盘总是不够用。

试过市面上各种清理工具：
- **CCleaner** - 被曝数据收集，不敢用了
- **Wise Disk Cleaner** - 广告越来越多
- **360安全卫士** - 臃肿，而且谁知道它在后台干什么
- **微软电脑管家** - 清理力度太弱

它们要么**隐私风险**，要么**广告捆绑**，要么**清理不彻底**。

最让我受不了的是：**我不知道它们到底删了什么**。

于是我决定自己写一个。

---

## CDrive_DeepCleaner 诞生

这是一个**开源、透明、无捆绑**的C盘深度清理工具。

### 核心特性

**全方位清理**
- Windows临时文件
- Windows更新缓存
- 浏览器缓存（Chrome、Edge、Firefox、Opera、Brave）
- 开发缓存（node_modules、pip、npm、huggingface）
- AI工具缓存（Cursor、VSCode、Trae、Ollama）
- 聊天软件缓存（微信、QQ、Telegram、Discord）
- 媒体软件缓存（Spotify、网易云音乐）
- 系统日志

**安全机制**
- 文件移到回收站，可恢复
- 系统关键文件自动保护
- 受保护文件类型不删除（.sys、.dll、.exe等）
- 危险路径警告

**用户体验**
- 现代化GUI界面
- 分类标签页展示
- 实时扫描进度
- 磁盘使用监控

## 技术亮点

### 针对中国用户优化

这是最大的差异化。国外工具根本不知道：
- 微信缓存在哪里
- QQ的临时文件在哪
- 网易云音乐的缓存怎么清理
- Trae、Cursor这些新IDE的缓存位置

这个工具专门针对中国用户的软件生态做了适配。

### 针对开发者优化

作为开发者，我们的C盘有大量：
- `node_modules` 文件夹
- `__pycache__` 缓存
- pip、npm、yarn 缓存
- HuggingFace 模型缓存
- 各种IDE的缓存

这个工具能精准找到并清理这些。

### 安全第一

- **回收站保护**：删除的文件先移到回收站，误删还能找回
- **系统文件保护**：pagefile.sys、hiberfil.sys等关键文件自动隐藏
- **危险路径警告**：Windows目录、System32等敏感区域拒绝删除
- **受保护扩展名**：.sys、.dll、.exe等文件类型不删除

## 使用方法

### 方式一：直接运行exe（推荐）

下载打包好的exe文件，双击运行即可。

### 方式二：Python运行

```bash
git clone https://github.com/dazhuang025-png/CDrive_DeepCleaner.git
cd CDrive_DeepCleaner
pip install -r requirements.txt
python main.py
```
